# Civic-Pulse
Vibe2Ship Hackathon
CivicPulse turns scattered complaints about potholes, water leaks, and broken streetlights into a transparent, trackable pipeline. Citizens report issues with a photo and location pin, AI auto-categorizes and prioritizes each one, and nearby residents verify it's real through upvotes. Every report moves through a public status timeline Reported to Resolved while ward-level dashboards expose resolution speed and department performance, replacing untracked complaints with real accountability.
